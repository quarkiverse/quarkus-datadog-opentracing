== Installation

If you want to use this extension, you need to add the `io.quarkiverse.opentracing.datadog:quarkus-datadog-opentracing` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.opentracing.datadog</groupId>
    <artifactId>quarkus-datadog-opentracing</artifactId>
</dependency>
----
